# 列表标签

- 无序列表

```html
<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    ...
</ul>
```

li：代表无序类表里的每一项

- 有序列表

```html
<ol>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    ...
</ol>
```

- ul、ol、li都是块级元素

#### 清除列表的默认样式

```css
ul,ol,li{
    list-style:none;
    margin:0;
    padding:0
}
```
